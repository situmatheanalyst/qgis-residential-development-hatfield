# 🏗️ GIS-Based Urban Planning Analysis

## 📍 Overview
This project presents a GIS-based spatial analysis of a proposed residential development located in Salisbury Village, Hatfield. The analysis applies geospatial techniques to evaluate land use distribution, residential density, accessibility to essential services, and transport network connectivity.

The objective is to demonstrate how spatial data can support informed urban planning and decision-making.

---

## 🛠️ Tools and Technologies
- QGIS (Geographic Information Systems)
- OpenStreetMap Data
- Spatial Analysis Techniques:
  - Buffer Analysis
  - Proximity Analysis
  - Land Use Mapping
  - Network Analysis

---

## 📊 Key Analyses

### 1. Relative Location
Assessment of the site’s position within Hatfield and its connectivity to surrounding urban infrastructure and road networks.

### 2. Existing Site Conditions
Evaluation of current land use patterns, infrastructure, and environmental context to determine development suitability.

### 3. Proposed Land Use
Design of a structured spatial layout incorporating residential zones, service areas, and green spaces.

### 4. Residential Density
Analysis of land use intensity through high, medium, and low-density classifications to support efficient spatial planning.

### 5. Proximity to Services
Accessibility assessment using buffer analysis to determine walking distance to essential services such as schools, retail, and healthcare.

### 6. Transport Network
Evaluation of internal and external connectivity through road hierarchy and transport infrastructure analysis.

---

## 🎯 Key Insights
- Strong integration with existing urban infrastructure  
- High accessibility to essential services  
- Efficient land use distribution  
- Well-structured transport network  
- Balanced approach to sustainability and livability
